Kobra - Flip Sensor Card

Dit is de flip-sensor-card voor Home Assistant. De kaart wisselt automatisch tussen meerdere sensoren, bijvoorbeeld binnentemperatuur en buitentemperatuur.

JS-bestand

Het JavaScript-bestand is beschikbaar via GitHub Pages: flip-sensor-card.js

Voorbeeld gebruik in Home Assistant


resources:
  - url: https://roy75.github.io/Kobra/flip-sensor-card.js
    type: module

type: custom:flip-sensor-card
entities:
  - sensor.woonkamer_temperatuur_2
  - sensor.buienradar_temperature
names:
  - Binnen
  - Buiten
interval: 3000
show_icons: true
icons:
  - mdi:home-thermometer
  - mdi:weather-sunny
icon_size: 2em

Opties

  • entities - lijst met sensor entity_idâ€™s
  • names - optionele namen voor elke sensor
  • icons - optionele mdi-iconen voor elke sensor
  • show_icons - true of false
  • icon_size - bv. 1.8em of 2em
  • interval - tijd in milliseconden tussen flips (standaard 3000)
